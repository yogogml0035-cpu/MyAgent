# Web Research Skill

## Workflow

1. Use the `tavily_search` tool to search for relevant information
2. Analyze the search results
3. Synthesize findings into a coherent response
4. Cite sources when possible

## Tips

- Use specific search queries for better results
- Set `topic="news"` for current events
- Set `topic="finance"` for financial data
