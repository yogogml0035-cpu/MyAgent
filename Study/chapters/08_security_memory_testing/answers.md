# 08 标准答案

## 自测题答案

1. 浏览器代码和 `NEXT_PUBLIC_*` 会暴露给用户，provider key 放前端会泄漏。后端负责读取 `.env` 和运行环境。
2. bug 修复、功能新增、交互改动、接口改动、状态流改动、前后端联动变化都需要对应浏览器 E2E；纯文档/纯注释除外。
3. 测试 runner 通常按命名发现测试；本仓库规范要求 Python 用 `test_*.py`，前端 Node 测试用 `test_*.test.ts`，避免测试被漏跑。
4. 会话上下文来自同一个 task 的历史消息和短期缓存，每次运行前确定性组装；长期记忆跨 task 检索，只保存脱敏后的稳定事实和偏好。

## 练习观察点

长期记忆的价值是“未来反复有用的稳定背景”，不是把当前对话、客户原文或一次性事实全存下来。
