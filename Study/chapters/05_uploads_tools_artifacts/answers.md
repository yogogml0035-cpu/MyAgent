# 05 标准答案

## 自测题答案

1. Upload 是用户上传到 task 工作区的源文件；Resource 是这些文件在工具层的受控表示；Artifact 是 Agent 运行后生成并供用户打开或下载的结果文件。
2. 大文档全部塞给模型会导致上下文膨胀、费用上升、结果不稳定，也可能暴露不必要的敏感内容。
3. HTML 产物可能包含不可信内容。直接 top-level 打开 blob 可能扩大脚本风险；项目用 sandbox iframe 禁用脚本来预览。
4. 前端校验能尽早给用户反馈；后端校验才是安全边界，不能信任浏览器传来的文件名和内容。

## 练习观察点

路径安全不是“看起来像相对路径”就够了，必须 `resolve()` 后确认目标仍在当前 task 的 `uploads/` 下。
